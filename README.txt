## Random Quote API

This application is a Flask-based API that serves random motivational quotes.
When you access `/quote`, it returns a random quote as a JSON response.

### How to run the app:
1. Pull the Docker image:
   docker pull Avrajk/random-quote-api

2. Run the app:
   docker run -p 5000:5000 Avrajk/random-quote-api

3. Open a browser and navigate to:
   http://localhost:5000/quote to see a random motivational quote.
